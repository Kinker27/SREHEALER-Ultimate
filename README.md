# 🌌 SREHEALER // ULTIMATE_V6
### *Autonomous Neural Maintenance & Self-Healing Engineering Suite*

**SREHEALER** is a revolutionary AI-driven agent designed to shift software maintenance from reactive debugging to autonomous healing. By combining **RAG-based project awareness** with an **Agentic Self-Correction Loop**, it doesn't just suggest fixes—it verifies them in a secure sandbox before deployment.

---

## 🚀 Revolutionary Features

* **🧠 Agentic Self-Correction:** The AI test-runs its own proposed fixes in a background sandbox. If it fails, the agent sees the error and heals itself until the code is **✓ VERIFIED**.
* **📂 Multi-File RAG Engine:** Unlike standard AI, SREHEALER scans your entire local project directory to ensure cross-file dependency logic remains intact.
* **🛡️ Security Shield:** A dedicated mode to hunt and patch **OWASP Top 10** vulnerabilities like SQL Injection and hardcoded secrets.
* **📊 Debt Analytics:** Real-time **Technical Debt Burndown Charts** and audit logs powered by SQLite to track codebase health over time.
* **✨ Cinematic DX:** A high-fidelity, glassmorphism interface designed for the modern developer experience.

---

## 🛠️ Tech Stack

- **AI Core:** Gemini 2.5 Flash (Neural Reasoning)
- **Backend:** Python, Flask, SQLite3
- **Frontend:** HTML5, CSS3 (Cinematic Animations), JS (Async), Chart.js
- **Execution:** Subprocess-based Sandboxing
